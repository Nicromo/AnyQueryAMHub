🔌 Chrome-расширение AM Hub Sync

Расширение синхронизирует данные из Merchrules в AM Hub прямо из браузера — каждые 30 минут автоматически. Установите его на каждом компьютере менеджера.

Вставьте этот токен в настройки расширения вместе с URL хаба
Инструкция установки:
1. Скачайте расширение: amhub-sync.zip
2. Откройте Chrome → chrome://extensions
3. Включите «Режим разработчика» (правый верхний угол)
4. Нажмите «Загрузить распакованное» → выберите папку расширения
5. Кликните иконку расширения → вставьте URL хаба и токен → сохраните
✅ Скопировано
