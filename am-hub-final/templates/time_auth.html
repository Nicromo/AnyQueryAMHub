Рекомендуется

Создаётся один раз в настройках Time — не истекает. Это правильный способ для интеграций.

1
Откройте time.tbank.ru → войдите через SSO Т-Банка (логин → пароль → SMS)
2
Нажмите на аватар (верхний правый угол) → Account Settings
3
Вкладка Security → раздел Personal Access Tokens → кнопка Create Token
4
Дайте название amhub → нажмите Create → скопируйте токен и вставьте ниже
🌐 Открыть Time

Запасной вариант

Если PAT недоступен. Cookie MMAUTHTOKEN истекает через ~24 часа — придётся обновлять каждый день.

1
Откройте time.tbank.ru и войдите через SSO
2
Нажмите F12 → вкладка Application (Chrome) или Storage (Firefox)
3
Слева: Cookies → https://time.tbank.ru → строка MMAUTHTOKEN → скопируйте значение из колонки Value
